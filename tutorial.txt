Passos para instalar o NASM:
1. Instalar o NASM:
Utilize o comando sudo apt install nasm no terminal para instalar o NASM. Este comando instala o assembler NASM e suas ferramentas relacionadas. 
2. Verificar a instalação:
Após a instalação, você pode verificar se o NASM foi instalado corretamente digitando nasm -v no terminal. Isso mostrará a versão do NASM instalada. 
3. Instalar ferramentas adicionais (opcional):
Para compilar e linkar código assembly, você pode precisar instalar outras ferramentas, como o compilador C (gcc) e o vinculador (ld). Utilize sudo apt install build-essential para instalar um conjunto de ferramentas de desenvolvimento essenciais, que inclui o gcc e o ld. 
4. Compilar código assembly:
Depois de instalar as ferramentas necessárias, você pode compilar um arquivo assembly (.asm) usando o comando nasm -f elf64 conversorDECparaHEX.asm -o conversorDECparaHEX.o (para um arquivo de saída ELF de 64 bits). 
5. Linkar o código:
Em seguida, utilize o comando ld -m elf_x86_64 conversorDECparaHEX.o -o conversorDECparaHEX para linkar o arquivo objeto gerado e criar um arquivo executável. 
6. Executar o código:
Finalmente, execute o arquivo executável utilizando ./conversorDECparaHEX
